# v1.0 Dissertation Submission

This release contains the initial research portfolio version of:

Grey-Zone Sovereignty and Market Adaptation:
Financial Responses to Taiwan-Related Geopolitical Shocks.

The repository includes the dissertation manuscript, event-study framework, data organization, analysis scripts, figures, and supporting research documentation.

The project investigates how Taiwan-related geopolitical shocks affect financial markets through investor interpretation, adaptation, and grey-zone sovereignty dynamics.
